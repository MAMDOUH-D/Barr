# بارّ 🤍
تطبيق رعاية الوالدين — يتيح لأفراد العائلة متابعة الأدوية والوجبات بشكل مشترك.

---

## رفع المشروع على GitHub وVercel

### الخطوة 1 — رفع على GitHub
1. افتح **github.com** وسجل دخول
2. اضغط **New repository**
3. اسم المشروع: `barr`
4. اضغط **Create repository**
5. ارفع جميع الملفات على الـ repository

### الخطوة 2 — ربط Vercel
1. افتح **vercel.com** وسجل دخول بحساب GitHub
2. اضغط **Add New Project**
3. اختر مشروع `barr`
4. اضغط **Deploy** — بدون أي إعدادات إضافية

### الخطوة 3 — تثبيت التطبيق على الجوال (PWA)
**iPhone:**
1. افتح الرابط في Safari
2. اضغط زر المشاركة (Share)
3. اختر **إضافة إلى الشاشة الرئيسية**

**Android:**
1. افتح الرابط في Chrome
2. اضغط **تثبيت التطبيق** أو من القائمة
